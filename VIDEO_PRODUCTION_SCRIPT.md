**Location:** Dimly lit room/studio with laptop

## üí° PRODUCTION TIPS & TRICKS

### Shooting Tips
1. **Stabilization:** Use tripod or gimbal, avoid handheld shakiness
2. **Focus:** Manual focus when possible to avoid hunting
3. **White Balance:** Lock white balance to avoid color shifts
4. **Overlapping Action:** Shoot 2-3 seconds extra before/after each take for edit flexibility
5. **Monitor Audio:** Use headphones to check for background noise

### Screen Recording Tips
1. **Hide Desktop Icons:** Clean up desktop before recording
2. **Close Notifications:** Turn on Do Not Disturb mode
3. **Cursor Visibility:** Install cursor highlighter for better visibility
4. **Browser Extensions:** Disable or hide extension icons
5. **Zoom Level:** Keep browser at 100% zoom
6. **Recording Area:** Set to full screen or specific window

### Editing Shortcuts
1. **Markers:** Add markers at key beats in music for timing
2. **Adjustment Layers:** Use for color grading multiple clips at once
3. **Nested Sequences:** Group screen recordings for easier organization
4. **Keyboard Shortcuts:** Learn J-K-L for playback speed
5. **Auto-Save:** Enable auto-save every 5 minutes

### Budget-Friendly VFX
1. **Glows:** Duplicate layer, add Gaussian blur, set to Screen blend mode
2. **Particles:** Use free stock footage from Pexels/Mixkit, composite with Screen blend
3. **Text Animation:** Use built-in title templates, customize colors
4. **Transitions:** Use simple cross-dissolve, avoid cheesy wipes
5. **Speed Ramps:** Add slow-mo for emphasis (48s card reveal)
